# Employee Attrition Dashboard (Power BI)

## 📊 Project Overview
This project is a Power BI dashboard designed to analyze employee attrition data and identify key factors affecting employee turnover.

##  Features
- Attrition rate analysis
- Department-wise and role-based insights
- Salary and experience impact on attrition
- Interactive filters and slicers

## 🛠️ Tools & Technologies
- Power BI
- Data Analysis
- Data Visualization

## 📈 Insights
- Identified high attrition departments
- Analyzed impact of salary and job role
- Highlighted employee retention patterns

##  File
- emp_attrition_dashboard.pbix

##  Purpose
This project demonstrates HR analytics and business intelligence skills using Power BI.
